## Task Manager Filter(React) ##
A real-world Task Management application built using React + Vite, designed step-by-step to cover core and advanced React concepts used in product companies.

## Features Implemented ##
# Core Features ##

Add new tasks
Delete tasks
Persistent storage using localStorage
Tasks remain after page refresh
Update Task 

# Advanced Features (Step 7) ##

Mark tasks as Completed
Inline Edit task
Strike-through UI for completed tasks
Clean, real-world state management

# React Concepts Covered ##
This project intentionally covers important React interview & real-world topics:
Functional Components
useState & useEffect
Controlled Inputs
Conditional Rendering
Immutable State Updates
Component Reusability
localStorage integration
Inline editing UX patterns
Filter tasks (All / Active / Completed)

## Tech Stack ##
